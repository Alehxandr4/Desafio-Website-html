Client-side e servers-side

Lado do cliente e lado do servidor são termos de desenvolvimento da web que descrevem onde o código do aplicativo é executado.

O modelo Client-Servers funciona a partir da comunicação do dispositivo do usuário a uma rede de servidores localizados, a onde iram obter os dados que procuram.

O dispositivo cliente (notebooks, smartfone e computadores) enviam solicitações de páginas web ou aplicativos aos servidores e os servidores fornecem a resposta procurada.

O cliente-side é a parte visual, é a onde se exibi os textos, imagens, as ações que o aplicativo ou o navegador executa para o usuário.

O serve-sede é o local que os processos são executados, é de onde vêm as informações que os usuários estão a procurar.

O que é um navegador?

Um navegador (browser) é um programa que possibilita que os clients a interagem com documentos HTML hospedados em servidores Web.

Eles exibem essas informações coletadas em nossos dispositivos, exemplos de navegadores são:

  • Navegador Google Chrome
  • Navegador Microsoft Edge
  • Navegador Mozilla Firefox

aplicação e Softwares

Aplicação Web, é um software que roda na internet, em vez de funcionar em um sistema operacional. Este software trabalha para servir algo ao navegador

Softwares desktopDiferent de aplicações web, estes Softwares precisam que seus códigos sejam enviados, instalados e executados diretamente em nossas máquinas e podem até usar dados compatilhados por um servidor web.

A aplicação mobile, tem um software muito parecido como do desktop, com a diferença que seus aplicativos se instala no smartfone vêm de uma loja de aplicativos do sistema operacional..

Estrutura básica de HTML

  1. O < head > elemento contém metainformações sobre a página HTML
  2. O < title > elemento especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página)
  3. O < body > elemento define o corpo do documento e é um recipiente para todos os conteúdos visíveis, como cabeçalhos, parágrafos, imagens, hiperlinks, tabelas, listas etc.
  4. O < h1 > elemento define o texto grande
  5. O < p >elemento define páragrafo

Elementos e atributos da HTML

  • Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final.
  • Alguns elementos HTML não possuem conteúdo. Esses elementos são chamados de elementos vazios.Elementos vazios não possuem uma tag final
  • Os elementos HTML podem ser aninhados (isso significa que os elementos podem conter outros elementos).
  • Os atributos HTML fornecem informações adicionais sobre os elementos HTML.

Atributos da HTML

  • Os elementos HTML podem ter atributos
  • Os atributos fornecem informações adicionais sobre os elementos
  • Os atributos são sempre especificados na tag de início
  • Os atributos geralmente vêm em pares de nome/valor como:name="value"
